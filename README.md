# Run in development

1. Clone the repository
2. Execute 
```
yarn install
```
3. Must have Nest CLI installed
```
npm i -g @nestjs/cli
```

4. Raise the database
```
docker-compose up -d
```


## Stack used
* MongoDB
* Nest
